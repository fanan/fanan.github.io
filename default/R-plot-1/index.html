R plot 1 first things first

绘图功能是R的杀手锏。和一些下拉框式的统计软件不同，R的绘图需要自己动手。

网络上关于R的绘图的文章已有成千上万，在此将自己日常工作要用到的一些绘图功能做一记录。

首先是最简单的在quartz或者X11下作图。

x <- c(1:10, 10:1)
plot(x, type = "p")

将图保存到pdf文件中。（保存为jpeg、png等格式的方法类似）

pdf("pdf_filename.pdf")
x <- c(1:10, 10:1)
plot(x, type = "n")
lines(1:length(x), x)
dev.off()

使用渐变色作图

svg("output.svg", width=10)
#png("1.png")
df = read.table("system_monitor.log")
loadavg = df$V2
timeFormat = "%Y-%m-%D-%H:%M:%S"
times = as.POSIXct(strptime(df$V1, timeFormat))
n = length(loadavg)
idx = as.integer(1:100 * (n * 0.01))
y = c(min(loadavg), loadavg, min(loadavg))
x = c(1,1:n,n)
plot(x, y, type="n", xlab="Seconds", ylab="load average", panel.first=grid(), xaxt = "n")
for (i in seq(255, 0, -3)) {
    #yy = c(0, ((loadavg - min(loadavg)) * (1 - i / 255)), 0)
    yy = y - (y - min(loadavg)) * (1 - i / 255)
    polygon(x, yy, col=rgb(1, i/255, 0), border=NA)
}
axis(1, at=x, labels=df$V1[x])
dev.off()
